# tingting

A private, bilingual static photographic exhibition built with plain HTML, CSS, and JavaScript.

## Replace photographs

1. Put full-size JPG files in `assets/full/`.
2. Put matching WebP previews in `assets/thumbs/`.
3. Update the bilingual `photographs` array in `script.js`.

Open `index.html` directly to view the exhibition. No backend or build step is required.
